# 2020-1 마이크로컴퓨터실험 텀프로젝트
## Board
- Name : TPK-128
- Device : ATmega128
- Clock setting : F_CPU = 14745600L
## Tool
- Atmel Studio 7.0
## Function
- Timer
- A/D Converter
- Game
## Port map
- DDRA = PA0 ~ PA7 (FND data)
- DDRB = PB0 ~ PB3 (FND Q)
- DDRC = PC4 ~ PC6 (LCD RS, R/W, E)
- DDRD = PD0 ~ PD7 (LCD Databus)
- DDRE = PE0 ~ PE1 (USART), PE4 ~ PE7 (Switch)
- DDRF = PF0 ~ PF3 (ADC)
